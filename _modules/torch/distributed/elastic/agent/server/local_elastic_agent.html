本地弹性代理(SimpleElasticAgent):
    torchelastic.agent.server.ElasticAgent 的实现，用于处理主机本地工作者。

此代理在每个主机上部署，并配置为启动 ``n`` 个工作者。
当使用 GPU 时，``n`` 对应主机上可用的 GPU 数量。

本地代理不与其他主机上部署的其他本地代理通信，即使工作进程可能进行跨主机通信。工作进程 ID 被解释为本地进程。代理作为一个整体启动和停止所有工作进程。
